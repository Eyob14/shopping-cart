## Requirements

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

## Setup

Install dependencies:
```bash
npm install
```

## Run tests

Run all tests:
```bash
npm test
```

## Project Structure

- `src/domain/` - Domain models (Product, CartItem, ShoppingCart)
- `src/utils.ts` - Utility functions (rounding)
- `tests/` - Test files
